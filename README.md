## Install the project:

To use the project, install the firebase SDK: 
```
https://github.com/firebase/firebase-ios-sdk.git
```

Add the following dependencies:

```
Firebase Analytis 
Firebase Auth 
Firebase Firestore 
Firebase Firestore Swift
Firebase Messaging 
Firebase Storage 
```
